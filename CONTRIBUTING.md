# Cómo contribuir a OpenBorme

¡Gracias por tu interés! Buscamos ayuda para mejorar la precisión de los parsers y la documentación.

## 🐛 Reportando Errores

Si encuentras un error de extracción o un enlace roto:

1. Abre un [Issue](https://github.com/PabloCirre/OpenBorme/issues).
2. Usa la etiqueta `bug`.
3. Proporciona el ID del acto afectado (ej: `BORME-A-2026-3024`).

## 🛠️ Desarrollo Local

1. Haz un Fork del repositorio.
2. Crea una rama para tu feature: `git checkout -b feature/mejora-ocr`.
3. Asegúrate de que tus cambios pasan los tests locales (si existen).
4. Envía un Pull Request describiendo tus cambios.

## 📜 Estándares de Código

* **PHP**: Sigue PSR-12.
* **Python**: Sigue PEP 8.
* **Commits**: Usa [Conventional Commits](https://www.conventionalcommits.org/) (ej: `feat: añade soporte para Girona`).

---
Al contribuir, aceptas que tu código se publique bajo la licencia MIT del proyecto.
